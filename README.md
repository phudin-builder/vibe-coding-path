# 🚀 Vibe Coding Path - Journey to AI Engineering

Chào mừng bạn đến với kho lưu trữ lộ trình học tập và phát triển kỹ năng **Vibe Coding** của tôi. Đây là nơi tôi ghi lại quá trình chinh phục việc tự động hóa quy trình (Workflow Automation) và xây dựng SaaS Prototype bằng sức mạnh của AI.

> **Thương hiệu cá nhân:** [Phu’s Builder Corner](https://tiktok.com/to-be-create) (Piano & Tech)

## 🎯 Mục tiêu dự án
Dự án này được xây dựng nhằm:
* Hệ thống hóa các bài tập từ khóa học **Mastering Vibe Coding**.
* Lưu trữ các công cụ tự động hóa (Automation Tools) tự xây dựng.
* Xây dựng Portfolio chuyên nghiệp để hướng tới vị trí **AI Quality Analysis** và **Data Analytics**.
